# PFI-Codigos
En el presente repositorio se encuentran los códigos utilizados en el Proyecto Final Integrador denominado 
'Desarrollo de un Sistema de Inspección Automatizada Basada en Imágenes para Redes de MT mediante Técnicas de Aprendizaje Automático'

La documentación y pasos de cada código se encuentra previo a cada paso

-El archivo denominado 'Anomalias_vs_No_Anomalias.ipynb' se trata del código utilizado para entrenar el modelo binario            
-El archivo denominado 'Anomalias_final.ipynb' se trata del código utilizado para entrenar el modelo multiclase
